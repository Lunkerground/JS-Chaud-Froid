Le Chaud / Froid !

Le but du jeu est simple ! L'ordinateur choisira un nombre compris entre 0 et 100, à vous de deviner lequel ! Pour cela, quelques indices vous seront donnés (je dis bien quelques !) afin de vous aider au long du jeu :

  • Chaud : vous vous approchez de la bonne réponse ;
  • Froid : vous vous éloignez de la bonne réponse ;

Pour commencer, vous devrez rentrer une valeur dans le champ de saisie ci-dessous puis cliquer sur le bouton "Valider" pour ... valider votre réponse (vous l'aviez pas vu venir celle-là) !

Bon jeu !
